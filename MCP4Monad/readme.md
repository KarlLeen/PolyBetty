### 项目描述
[MCP4Monad](https://github.com/LikaiTOP/MCP4Monad)
本项目是一个基于 Model Context Protocol (MCP) 的服务端示例项目，演示了如何通过 MCP 协议为 AI/LLM 工具（如 Cursor）提供 Monad 区块链测试网的链上数据查询能力。
